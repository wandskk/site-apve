<!-- GETTING STARTED -->
## Getting Started
1. Clone the repo
 ```sh
   git clone https://github.com/wandskk/site-apve.git
 ```
2. Go to project folder
 ```sh
   cd site-apve
 ```
3. Install project requires
 ```sh
   npm install or npm i
 ```
4. Run gulp watch
```sh
   gulp wach
 ```
